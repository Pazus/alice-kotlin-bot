# Implementation
* Add the JitPack repository to your **root** build.gradle file:
```
repositories {
    maven { url "https://jitpack.io" }
}
```
* Add dependencies to your **module** build.gradle file:
```
dependencies {
    implementation 'com.github.SavinMike:alicebot:v0.1.0'
}
```
